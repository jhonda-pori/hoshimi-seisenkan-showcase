# 収益化モデル

## 概要

ホシミ占星館は**クレジット制 + 広告**のハイブリッドモデルを採用。基本機能を広告視聴で無料提供し、より深い鑑定をクレジット購入で収益化しています。

## 2段階の鑑定モデル

| 段階 | 鑑定者 | 料金 | 内容 |
|------|--------|------|------|
| **無料鑑定** | ウララ（見習い） | 広告視聴 | 軽めの鑑定文（Gemini Flash Lite） |
| **本格鑑定** | ホシミ先生（師匠） | 2クレジット | 深い鑑定文（Gemini Pro） |

無料の鑑定で占いの体験価値を感じてもらい、より深い鑑定への課金を促す設計です。

## クレジット制

### クレジットパック

| アイテム | クレジット数 | 価格 |
|----------|-------------|------|
| 2クレジット | 2 | ¥160 |
| 10クレジット | 10 | ¥600 |
| 30クレジット | 30 | ¥1,500 |

まとめ買いほどお得な価格設定。

### クレジット消費

| アクション | 消費量 |
|-----------|--------|
| ホシミ先生の鑑定（算命学） | 2クレジット |
| ホシミ先生の鑑定（タロット） | 2クレジット |
| 追加質問チャット | 1クレジット/回 |
| キャッシュヒット時の鑑定 | 0クレジット（無料） |

## 広告収益

### リワード広告

ウララの無料鑑定は、リワード広告を視聴することで利用可能:

```
鑑定結果表示
  │
  ├── 「ウララの鑑定を聞く」ボタン
  │
  ▼
リワード広告を視聴
  │
  ▼
広告トークン（ワンタイム）を取得
  │
  ▼
鑑定文を生成・表示
```

### バナー広告

無料ユーザーに対してアプリ内にバナー広告を表示。

### 広告除去

買い切り型の広告除去アイテム（ad_free）を購入すると、すべての広告が非表示に。


## 購入検証フロー

アプリ内課金のセキュリティ:

```
ユーザーがアイテムを購入（Google Play / App Store）
  │
  ▼
クライアントが purchase_token / transaction_id をサーバーに送信
  │
  ▼
サーバーがプラットフォームAPIで購入を検証
  ├── Google Play: Developer API で purchaseState を確認
  └── App Store: Server API でJWT署名を検証
  │
  ▼
検証成功 → クレジット付与 or 広告除去フラグ設定
  │
  └── purchase_token を記録（冪等性保証）
```
